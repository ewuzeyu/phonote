# Plan: Remove unnecessary location permissions

## What
Remove `ACCESS_FINE_LOCATION` and `ACCESS_COARSE_LOCATION` from `AndroidManifest.xml`.

## Why
- No code uses location APIs
- IP detection uses UDP socket trick, NetworkInterface, and ConnectivityManager — none need location permission
- These permissions trigger a permission dialog on install, confusing users

## File to modify
`app/src/main/AndroidManifest.xml` — delete the two `<uses-permission>` lines.
